# Water-Pump-Control-with-esp32
As the title suggests, this project uses an ESP32 to control a water pump for an automated plant irrigation system. It was designed to water my plants automatically based on a predefined schedule while also providing manual control through a web-based dashboard.

To make the system more intelligent, I integrated several sensors along with the OpenWeather API, allowing the irrigation logic to adjust the amount of water dispensed according to the current temperature and weather conditions.

For a detailed explanation of the system architecture, hardware components, software implementation, and development process, visit my Medium blog.
